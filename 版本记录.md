# ImageViewer 版本记录

---

## v1.3.5 — 2026-03-11
**缩略图 + 工具栏浮层化**
- 缩略图栏和工具栏改为浮层叠加在图片区域底部（移出独立 Grid 行），渐隐后图片占满全高，不再遮挡
- 智能展示模式下工具栏与缩略图同步渐显 / 渐隐（统一动画 `BottomPanel`）
- 修复旧版 Opacity=0 占行高导致"图片被遮挡"的问题

## v1.3.4 — 2026-03-11
**缩略图交互优化**
- 修复智能展示模式下点击缩略图导致缩略图栏消失的问题（`ShowImage` 仅在 `Collapsed` 时才调用 `PrepareSmartThumb`，不打断已显示状态）
- 当前图片缩略图改为居中显示；左右导航时自动居中跟随（`ScrollThumbIntoView` 改为居中算法，`ActualWidth=0` 时延迟到 `Loaded` 优先级执行）
- 非当前缩略图鼠标悬停时高亮背景（`#3A3A3A`），离开后恢复（`#2A2A2A`）；切换选中时重置所有悬停背景

---

## v1.3.3 — 2026-03-11
**Toggle 开关改为圆点滑块样式**
- 缩略图栏 / 智能展示开关从色块改为胶囊滑块：34×18 圆角胶囊 + 14×14 白色圆点
- 圆点通过 `Canvas.Left` 动画（160ms CubicEase EaseInOut）从左（2px）滑动到右（18px）
- 胶囊背景颜色随状态切换：关闭 `#3A3A3A`，开启 `#0078D4`
- 新增 `AnimateToggle(pill, dot, isOn)` 统一处理两个开关的动画

## v1.3.2 — 2026-03-11
**智能展示热区修正（鼠标命中）**
- 修复智能模式下缩略图区域无法触发渐显的问题
- 根本原因：`Visibility=Collapsed` 时元素不参与鼠标命中测试
- 智能模式下改为 `Opacity=0 + Visibility=Visible`（透明但占位），热区事件正常触发
- 新增 `PrepareSmartThumb()`：加载图片后使缩略图栏处于可触发状态
- `ClearImageState()` 改为直接 Collapsed，不受智能模式影响
- `FadeOutThumbStrip()` 淡出后不再 Collapse，保持热区有效

## v1.3.1 — 2026-03-11
**智能展示热区范围**
- 热区改为缩略图栏 + 工具栏（图片区域不再触发渐显）
- 移除 ContentGrid 的 MouseEnter/MouseLeave，工具栏 Border 加入热区事件

---

## v1.3 — 2026-03-11
**弹出菜单重设计 + 缩略图模式**
- 更多按钮弹出菜单改为自定义 Popup：深色背景 `#2C2C2C`、圆角 8、DropShadowEffect
- 新增 `PopupRowStyle`：圆角 5、悬停 `#22FFFFFF`、`HorizontalContentAlignment=Stretch`
- 菜单新增**缩略图栏**开关（常驻显示）和**智能展示**开关（两者互斥）
- 开关状态用 22×12 圆角指示点显示：激活蓝 `#0078D4`，关闭灰 `#3A3A3A`
- 智能展示：鼠标进入热区渐显缩略图（180ms），离开后 150ms 延迟渐隐（200ms）
- `FadeInThumbStrip` / `FadeOutThumbStrip` 动画；`EnsureHideTimer` 防切换闪烁

## v1.2.1 — 2026-03-11

- 信息按钮（ℹ）移至缩放输入框左侧

- 移除删除按钮
- 缩略图栏改为无滚动条（Hidden），未打开图片时隐藏

## v1.2 — 2026-03-11
**缩略图栏 + 工具栏重构**
- 新增底部缩略图栏（88px），两阶段异步加载，`CancellationToken` 支持切换取消
- 缩略图点击跳转，当前项蓝色高亮，导航时自动滚动到可见区域
- 缩放%输入框移至工具栏左侧
- OpenFileDialog Filter 改为从 `SupportedExts` 自动生成
- 删除后 UI 重置提取为 `ClearImageState()` 方法

## v1.1 — 初始版本
**基础图片浏览器**
- 自定义无边框窗口（深色主题）
- 打开文件 / 拖放 / 文件夹扫描，同目录图片自动列表
- 左右导航、键盘翻页（←→ / PageUp/Down）
- ZoomBorder 缩放平移、缩放%手动输入框
- 旋转（左/右 90°）
- 图片信息弹窗、删除、在资源管理器中显示、复制路径
- Ctrl+拖拽导出到其他程序
