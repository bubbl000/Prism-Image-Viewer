# ImageViewer

WPF 图片浏览器，支持多格式浏览、缩略图、缩放平移、旋转、右键菜单、文件监视等。

## 功能特性

- 打开图片 / 拖放文件或文件夹
- 同目录图片自动列表，左右键 / 方向键翻页
- 缩放平移（滚轮缩放 + 鼠标拖拽），支持手动输入缩放百分比
- 旋转（左 / 右 90°）
- 底部缩略图栏，点击跳转，异步加载
- 图片信息**独立浮动窗口**（分辨率、大小、路径、修改时间）
- Ctrl + 拖拽导出到其他程序，**超出窗口 40px 自动切换跨软件拖拽**
- 深色无边框自定义窗口
- **右键菜单**：重命名、复制、另存为、资源管理器中打开、复制文件路径、设为壁纸、删除、打印
- **文件监视**：自动检测图片被删除/重命名，实时同步界面
- **文件夹穿透**：到达边界时可进入相邻文件夹，支持循环
- **设置窗口**：标题栏汉堡菜单 ≡，统一管理缩略图栏、智能展示、鸟瞰图、文件夹穿透

## 支持格式

JPG / JPEG / PNG / BMP / GIF / TIFF / TIF / WEBP / PSD / PSB / RAW
