# ImageViewer

WPF 图片浏览器，支持多格式浏览、缩略图、缩放平移、旋转等。

## 功能特性

- 打开图片 / 拖放文件或文件夹
- 同目录图片自动列表，左右键 / 方向键翻页
- 缩放平移（滚轮缩放 + 鼠标拖拽），支持手动输入缩放百分比
- 旋转（左 / 右 90°）
- 底部缩略图栏，点击跳转，异步加载
- 图片信息弹窗（分辨率、大小、路径、修改时间）
- 在资源管理器中定位 / 复制文件路径
- Ctrl + 拖拽导出到其他程序
- 深色无边框自定义窗口

## 支持格式

JPG / JPEG / PNG / BMP / GIF / TIFF / TIF / WEBP
